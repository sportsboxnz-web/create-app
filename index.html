15-Minute Session
Launch Script
1
Open
0:00 — 0:45
⌄
Say This
"You've all seen the trailer. You've all heard the presentation. You know what CREATE is. Today is different — today we make it real. No theory. No slides. Just the six things we do with every customer, every time."
2
Six Non-Negotiables
0:45 — 2:45
⌄
Delivery Note Walk through each one. Pause after each. Let them land.
C
Connect Quickly
"Welcome to Tile Depot!" — within 30 seconds, every customer.
R
Really Listen
"Tell me about the project you're working on." — every engagement.
E
Explain Simply
Always translate technical terms into plain benefit language.
A
Add Value
All quotes must be generated using the website quote function.
T
Take Responsibility
Full quote read-back — name, size, details, quantity, availability, guarantee.
E
End Memorably
Always thank by name, reference their project, invite them back.
Say This
"These six are not suggestions. They are the standard. Every customer. Every time. If you're ever unsure what to do next — come back to these."
3
Why It Matters
2:45 — 3:30
⌄
Say This
"Customers don't remember tile specs. They remember how they felt when they left. Confident. Cared for. Inspired. That's what CREATE delivers — and it's what sets us apart from every other tile store."
"Our competitors sell the same products. We sell a better experience. These six standards are how we prove it every single day."
4
Training Pathways
3:30 — 5:00
⌄

Training Portal

  • One module per CREATE step
  • Self-paced, online
  • Tracked across all stores

Micro Installs

  • 5-min weekly huddle
  • One behaviour per week
  • Cycles all six steps
Say This
"Today is the start, not the finish. You'll have Training Portal modules — one per CREATE step. And every week I'll run a five-minute micro install. One behaviour. One standard. Every week."
5
Choose Your Weapon
5:00 — 9:00
⌄
Delivery Note Read each option out. One sentence each. Team picks two. Don't push a preference.
1
Shift Starter Card
Laminated card at the counter. Manager sets one CREATE behaviour as the daily focus.
2
Floor Walk
5 minutes per shift. Catch one person doing CREATE well. Name it out loud.
3
Customer Story Debrief
One real customer story at end of shift. Manager maps it back to CREATE.
4
WhatsApp Nudge
One sentence to the group chat each morning. One behaviour. Ten seconds.
5
Scoreboard
Whiteboard tracking one CREATE metric per week. Visible to the whole team.
Say This
"Which two feel right for us? You know your store."
"Good. We're doing [Option X] and [Option Y]. Starting this week. I'll run both."
6
Set It Up
9:00 — 13:00
⌄
Delivery Note Both techniques activated before anyone leaves the room.
Option 1 — Shift Starter Card
Place the Reference Card at the counter now. Announce today's focus behaviour.
Option 2 — Floor Walk
Set a phone reminder now. Commit to a time. Name this week's watch behaviour.
Option 3 — Customer Story Debrief
Nominate who goes first. Agree on when. Set it now.
Option 4 — WhatsApp Nudge
Open the group chat now. Send the first message before the session ends.
Option 5 — Scoreboard
Grab a marker. Write this week's metric on the whiteboard. Put it up now.
Say This
"Both of these go live right now — not tomorrow, not next week. Before we close."
7
Close
13:00 — 15:00
⌄
Say This
"We've got six non-negotiables. We've chosen two ways to keep them alive. That's all CREATE needs — consistency and visibility."
"One last question: which CREATE standard, if you nailed it every single time, would make the biggest difference to your customers?"
"Your next customer is your first rep. Let's go."
Hand Out Distribute Reference Cards now. One per person.

Session Complete

Daily & Weekly
Operating System
Ask Yourself Every Shift
Which CREATE behaviour did I reinforce today?
☀️
Every Shift
Daily non-negotiables for managers
Run your chosen reinforcement technique — Shift Starter Card, Floor Walk, WhatsApp Nudge, Story Debrief, or Scoreboard.
Catch one person doing CREATE well. Name it out loud in front of the team.
If you hear a wrong phrase — correct it calmly and immediately. Reset to the standard.
📅
Every Monday
Weekly Micro Install — 5 minutes
Name the one CREATE behaviour you're focusing on this week.
Demonstrate bad vs best — 60 seconds, no roleplay required.
Have each person say the phrase out loud once. That's the install.
🏆
Every Friday
Recognition + behaviour check
Call out one CREATE win publicly — specific behaviour, specific person.
Complete the Traffic Light Behaviour Check for the week.
Identify the lowest-scoring letter — that's next week's focus.
📊
Weekly Manager Meeting
CREATE pulse — 5 minutes
Share one behaviour reinforced this week.
Share one CREATE win from your store.
Name your focus behaviour for next week.
🔄
6-Week Micro Install Rotation
One CREATE step per week
Week 1 — Connect Quickly: "Welcome to Tile Depot!"
Week 2 — Really Listen: "Tell me about the project..."
Week 3 — Explain Simply: feature → benefit translation
Week 4 — Add Value: website quote function + project thinking
Week 5 — Take Responsibility: full quote read-back
Week 6 — End Memorably: name + project + invitation
